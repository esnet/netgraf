# NetGraf: Real-time Network Monitoring Machine Learning Pipeline:


## Installation Guide

Install all the dependencies by running the following:

    sudo pip3 install -r requirements.txt


To Lauch your NetGraf ML GUI App, run the following:

    streamlit run app.py


To view the ML Prediction Dashboard on your browser:


     http://localhost:8501
